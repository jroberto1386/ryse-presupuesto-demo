# Ryse Presupuesto Demo (React + Vite + Tailwind)

Demo de control presupuestal 2026 para Ryse Médica.

## Requisitos
- Node.js 18+ y npm

## Uso
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## GitHub Pages
1) Ajusta `base` en `vite.config.js` a `'/<tu-repo>/'` si el repo no es de usuario/org.
2) Ejecuta:
```bash
npm run deploy
```

## Carga de catálogos / RLS
- En rol **Finanzas**, usa **Cargar config** y **Cargar RLS** para pegar JSON.
